# Требование

   1. Node.js.
   2. Gulp.
   3. Git.
   
# Описание запуска

   1. Клонирование проекта:
   ```sh
   git clone https://github.com/Yevheniishev/CRM.github.io.git
   ```
   2. Перейти в папку с проектом:
   ```sh
   cd CRM.github.io
   ```
   3. Установка npm-зависимостей:
   ```sh
   npm i
   ```
   4. Запустить Gulp:
   ```sh
   gulp
   ```
